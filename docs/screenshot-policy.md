# Screenshot Policy

Screenshots and other visual assets used in this repository, its documentation,
or any public store listing must follow this policy.

## Allowed

- Synthetic or redacted screenshots that show only the extension's own UI
  (popup, options page) or a clearly placeholder ChatGPT layout.
- Diagrams and mockups created for documentation.

## Prohibited

- Real ChatGPT conversations, messages, or titles.
- Real prompts or assistant responses.
- Real account names, email addresses, or profile images.
- Personal browser windows, bookmarks, or history.
- Local filesystem paths or OS usernames visible in a capture.
- Cookies, tokens, or authorization headers shown in any form.

## Handling

- Generate screenshots in a clean, logged-out-or-synthetic environment.
- Prefer the bundled synthetic fixture for any UI demonstration.
- If a real environment is used for verification, crop and redact before any
  public use, and never commit the original capture.
- Treat all personal screenshots as local-only and non-distributable.
